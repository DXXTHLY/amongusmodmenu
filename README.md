## ✨ Features

### 🗺️ Live Radar
Real-time map showing every player's position as it updates. Renders actual **character head sprites** instead of plain dots — and when a player is killed their sprite swaps to a **dead body** right at the location they were eliminated. Supports all three maps with a built-in switcher.

| Map | Supported |
|-----|-----------|
| The Skeld | ✅ |
| Polus | ✅ |
| MiraHQ | ✅ |

### 👥 Player ESP Table
Full lobby breakdown updating live with every player's **name**, **color** and **role**. Threats are always prioritized — impostors and their variants pin to the top of the list in red automatically.

| Role | Display |
|------|---------|
| Impostor / Shapeshifter / Phantom / Viper | 🔴 Pinned to top |
| Crewmate / Engineer / Scientist etc | ⚪ Middle |
| Dead / Guardian Angel | 🔘 Bottom |

### 📌 Overlay Mode
One click makes **143X Menu float on top of Among Us** at 85% opacity so you can read it mid-game without ever alt-tabbing. Click again to disable.

### 🔲 Compact Mode
Toggle the radar off and the window collapses to just the player list — slim, clean and out of the way. Toggle it back on whenever you need the map.
